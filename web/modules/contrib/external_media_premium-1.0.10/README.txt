INTRODUCTION
------------

This is a premium version of the External Media module with extra import
services such as Unsplash, Instagram, Pexels, Pixabay, AWS and any other external URL.

Webform module support and Media Library support.

 * For a full description of the premium module, visit the project page:
   https://downloads.minnur.com/drupal/external-media-widget

 * Drupal.org contrib page for the External Media module:
   https://www.drupal.org/project/external_media


REQUIREMENTS
------------

In order to be able to use external services installing appropirate SDKs
required such as:

 * composer require espresso-dev/instagram-basic-display-php

 * composer require unsplash/unsplash

 * composer require zoonman/pixabay-php-api

 Those SDKs usually gets installed with the contrib External Media module


INSTALLATION
------------

 * Install External Media contrib module in order to be able to use this
 premium module.
