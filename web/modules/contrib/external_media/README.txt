INTRODUCTION
------------

Popular Wordpress plugin with over 8000 active installations brought into
Drupal. Pick files from Dropbox, Box, Instagram, Google, Unsplash, AWS 
and many more. Import high quality free stock photos.

 * For a full description of the  module, visit the project page:
   https://www.drupal.org/project/external_media

 * Premium module with extra services:
   https://downloads.minnur.com/drupal/external-media-widget


REQUIREMENTS
------------

In order to be able to use external services installing appropirate SDKs
required such as:

 * composer require espresso-dev/instagram-basic-display-php

 * composer require unsplash/unsplash

 * composer require zoonman/pixabay-php-api


INSTALLATION
------------

 * Installing with composer is the best so it would automatically install
 all the required PHP SDKs.
